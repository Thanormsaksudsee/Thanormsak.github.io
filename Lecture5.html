Introduction to Functions

A function is a group of statements that exist within a program for the purpose of performing a specific task.

Introduction to Functions

Functions
• A function is a section of reusable code that performs an action.
• A function has a name and is called, or executed, by that name.
• Optionally, functions can accept arguments and return data.

FUNCTIONS NAMES

Function naming rules:
• ไม่สามารถใช้คำสำคัญเป็นชื่อฟังก์ชันได้
• ไม่สามารถมีช่องว่าง
• อักขระตัวแรกต้องเป็นตัวอักษรหรือขีดล่าง
• อักขระอื่นๆ ทั้งหมดต้องเป็นตัวอักษร ตัวเลข หรือขีดล่าง
• อักขระตัวพิมพ์ใหญ่และตัวพิมพ์เล็กต่างกัน

BENEFITS OF MODULARISING A PROGRAM WITH FUNCTIONS


• รหัสที่ง่ายกว่า: เข้าใจง่ายขึ้นเมื่อแบ่งออกเป็นฟังก์ชัน
• Code Reuse: เขียนครั้งเดียวเพื่อดำเนินการนั้นแล้วดำเนินการใดๆ เวลาที่จำเป็น
• การทดสอบที่ดีขึ้น: สามารถทดสอบแต่ละฟังก์ชันในโปรแกรมทีละรายการ แก้ไขข้อผิดพลาดได้ง่ายขึ้น
• การพัฒนาที่เร็วขึ้น: สามารถรวมไลบรารีของฟังก์ชันในแต่ละโปรแกรมได้ ที่ต้องการพวกเขา
• อำนวยความสะดวกในการทำงานเป็นทีมได้ง่ายขึ้น : โปรแกรมเมอร์ต่างๆ สามารถมอบหมายงานได้ ในการเขียนหน้าที่ต่างๆ
