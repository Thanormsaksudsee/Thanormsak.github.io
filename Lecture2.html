SELECTION IF ELIF ELSE

บทที่ 3 จะพูดถึงโครงสร้างการตัดสินที่เป็น1ในรูปแบบหนึ่งของการเขียนโปรแกรมที่มีความสำคัญเป็นอย่างมาก

DECISION STRUCTURE

 โครงสร้างการตัดสินใจคือโครงสร้างในโปรแกรมคอมพิวเตอร์ที่ช่วยให้โปรแกรมตัดสินใจและเปลี่ยนพฤติกรรมตามการตัดสินใจนั้น การตัดสินใจขึ้นอยู่กับผลลัพธ์ของการทดสอบเชิงตรรกะ การทดสอบเชิงตรรกะคือการคำนวณที่ผลลัพธ์เป็นจริงหรือเท็จ

IF

คำสั่ง if ถูกใช้เพื่อสร้างโครงสร้างการตัดสินใจ ซึ่งช่วยให้ a โปรแกรมให้มีมากกว่าหนึ่งเส้นทางของการดำเนินการ คำสั่ง if ทำให้เกิด คำสั่งอย่างน้อยหนึ่งคำสั่งเพื่อดำเนินการเฉพาะเมื่อนิพจน์บูลีนเป็นจริงเพื่อความง่าย เราจะอ้างถึงบรรทัดแรกเป็น if clause ประโยค if ขึ้นต้นด้วยคำว่า if ตามด้วยเงื่อนไข คือ an นิพจน์ที่จะประเมินว่าเป็นจริงหรือเท็จ ลำไส้ใหญ่ ปรากฏขึ้นหลังจากเงื่อนไข เริ่มต้นที่บรรทัดถัดไปเป็นบล็อกของ งบ. บล็อกเป็นเพียงชุดของคำสั่งที่เป็นของ กันเป็นกลุ่ม สังเกตในรูปแบบทั่วไปว่า . ทั้งหมด ข้อความในบล็อกถูกเยื้อง จำเป็นต้องเยื้องนี้ เพราะล่าม Python ใช้เพื่อบอกว่าบล็อกเริ่มต้นที่ใด และจบลง
